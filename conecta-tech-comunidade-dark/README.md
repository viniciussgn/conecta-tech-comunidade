# Conecta Tech Comunidade — versão visual inspirada no template enviado

Este projeto foi adaptado para GitHub Pages em HTML, CSS e JavaScript puro, mas com estética inspirada nos códigos React/Hostinger enviados.

## Estrutura
- `index.html`
- `style.css`
- `script.js`
- `pages/`
- `materiais/`
- `assets/`
- `data/noticias.json`

## Publicação no GitHub Pages
1. Crie um repositório chamado `conecta-tech-comunidade`.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Source**, escolha **Deploy from a branch**.
5. Selecione a branch principal e a pasta `/root`.
6. Salve e aguarde o link público.

## Observação
O site foi construído para ser leve e fácil de publicar, sem depender de React ou Vite no GitHub Pages.
